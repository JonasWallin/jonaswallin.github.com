Infinite dimensional AMCMC for Gaussian processes

Wallin, Vadlamani

(tech report )

We have develop adaptive versions of the preconditioned Crank–Nicolson Langevin (pCNL) algorithm, which can be viewed as an infinite dimensional version of the well known Metropolis-adjusted Langevin algorithm (MALA) algorithm for Gaussian processes. We also use adaptive methods tailored for infinite dimensions. The figure above shows the estimation of the eigenvalues if one uses infinite dimensional estimation methods or regular.
