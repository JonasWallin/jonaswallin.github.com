non-Gaussian Matern Fields

Bolin, Wallin, Lindgren

(submitted)

In this article, we exaim how to preform estimation and predicition on a certain type of continous non Gaussian random fields models. The article is an application extension of the article Bolin 2013 (now published in SJS). We extend the models in the article, by adding measurement noise and covariates; the estimation is done through a MCEM algorithm. We also test how the models preform on a real data set with precipitation over the US; comparing the model to the standard Gaussian and transformed Gaussian models. The conlusion for this data is that the models works well but we need to incorporate covariates into the variance, since the mean and variance of the data is clearly linked.

Extensions:

  • The next planned extension is to change from MCEM to other stochastic estimation methods; There are two main reasons for this, first hopefully we can improve the speed of the algorithm, secondly we hope it allowes for more advanced models.

  • Expanding the model for regular grids; where a more rich noise class can be studied.
