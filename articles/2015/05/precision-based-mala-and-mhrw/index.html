Precision based MALA and MHRW

File: AMCMC_precision.zip

This code is used to online estimate the Cholesky factor of a sparse precision matrix, used to efficently sample densities using MALA or MHRW.

Most of the code is written matlab. The online updating algorithms is written in C++ with openmp.

Also to examples where the aglorithms are used is included.
